# Adab Brand Kit Guide

This guide explains where to find and how to use all official Adab brand assets from `/admin/brand`.

## Brand constants

- Navy: `#1B2A4A`
- Gold: `#C9A227`
- Cream: `#F8F6F1`
- Email: `hello@adab.ng`
- Phone: `+2348128272287`
- CAC registration: `RC 9590252`

Do not place the CAC logo in email signatures or letterhead. Use registration text only.

## Where to download

Open `/admin/brand` and use these sections:

1. **Downloadable assets**: logo, favicon, social image, color reference
2. **Office templates**: letterhead, email signature, business card, property flyer
3. **Corporate kit (Word)**: editable `.doc` profile/listing/application forms
4. **Corporate presentation**: browser deck for meetings and PDF export
5. **Print kit**: t-shirt design, sticker sheet, jotting pad, banner roll-up, A5 flyer
6. **Digital cards**: digital card + vCard + QR flow

## Export instructions

### Word-editable `.doc` files

- Download from **Corporate kit (Word)**.
- Open in Microsoft Word, edit placeholders, and save as `.docx` or PDF.

### Corporate presentation deck

- Open `corporate-deck.html` from **Corporate presentation**.
- Present full-screen in browser, or press `Ctrl/Cmd + P`.
- Export as landscape PDF with background graphics enabled.

### Print kit files

- Open each HTML template in browser.
- Replace text placeholders and keep logo aspect ratio unchanged.
- Print using the indicated paper size or export to PDF for vendor production.

### Digital card + QR/vCard flow

- Share `digital-card.html` as the public contact card.
- `adab-contact.vcf` is the downloadable contact file.
- `qr-vcard-flow.html` provides QR usage flow and instructions for field teams.

## Governance

- Use only approved brand colors and official contact details.
- Keep RC number as `9590252` in all corporate-facing material.
- Route template updates through the web repo so `/admin/brand` remains the source of truth.
